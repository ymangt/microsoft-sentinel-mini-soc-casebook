# Phishing URL Response Checklist

## Initial Triage

1. Confirm the sender, recipient, subject, URL, and URL domain.
2. Determine whether the message was delivered, blocked, quarantined, or clicked.
3. Identify all recipients exposed to the same sender, subject, or domain.
4. Check whether the suspicious domain appears in the watchlist and why.

## Containment

1. If delivered, recommend removing or quarantining matching messages.
2. If clicked, recommend account review and sign-in investigation.
3. Recommend user notification if exposure is confirmed.
4. Recommend blocking the sender/domain only if evidence supports it.

## Documentation

Record:

- Sender
- Recipients
- URL/domain
- User action
- Scope of exposure
- Response recommendation
